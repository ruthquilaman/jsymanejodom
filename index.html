¿Qué es JavaScript?

JavaScript es un lenguaje de programación interpretado, de alto nivel, que se utiliza principalmente en el desarrollo web del lado del cliente. Permite crear interactividad dinámica en las páginas web.

Uso de la Consola del Navegador y console.log()

La consola del navegador es una herramienta esencial para depurar y entender el comportamiento de tu código JavaScript.

Cómo acceder a la consola del navegador:

  • En Chrome, Firefox, o Edge: Presiona F12 o Ctrl+Shift+I (Cmd+Option+I en Mac).

Una vez abierta la consola, selecciona la pestaña "Consola" si no está ya seleccionada.

Uso de console.log()

console.log() es una función que te permite imprimir mensajes en la consola del navegador. Es muy útil para depurar y entender el flujo de tu código.
